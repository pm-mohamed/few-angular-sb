# Technical Specifications

Most of the time we create our own components according to our needs, we will use third party libraries for more complex components in the future.

<br></br>

##  [Antdesign](https://ng.ant.design/docs/introduce/en)

Will be used mainly for the complex time and date input components:

> Date pickers

> Time pickers

> Range pickers

> Week pickers

> Progress bars (Bars and Circular)

##  [Angular CDK](https://github.com/angular/components)

For dragabble elements logic.

##  [Apex Charts[TBA]](https://apexcharts.com/docs/angular-charts/)

For charts.
